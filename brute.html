Step 2: Define the Brute-Force Function

Create a function to test the target password against the embedded wordlist.

def brute_force_cracker(target_password, max_attempts=None, progress_interval=10000):
    """
    Attempts to crack a password by matching it against an embedded wordlist.
    Args:
        target_password (str): The password to crack.
        max_attempts (int, optional): Maximum number of attempts before giving up.
        progress_interval (int): Print progress every N attempts.
    """
    attempts = 0
    start_time = time.time()
    
    if not target_password:
        print("[!] Error: Target password cannot be empty.")
        return

    print("[+] Starting brute-force attack on password using embedded wordlist...")
    print("[!] Legal Note: Use this tool only for educational purposes or with explicit permission.")

    try:
        for word in WORDLIST:
            attempts += 1
            
            if attempts % progress_interval == 0:
                print(f"[*] Attempts: {attempts}, Time elapsed: {time.time() - start_time:.2f}s")

            if max_attempts and attempts >= max_attempts:
                print(f"[!] Reached max attempts ({max_attempts}). Stopping.")
                return

            if word == target_password:
                elapsed_time = time.time() - start_time
                print(f"\n[+] Password cracked!")
                print(f"Password: {word}")
                print(f"Attempts: {attempts}")
                print(f"Elapsed time: {elapsed_time:.2f} seconds")
                return
                
        print("\n[-] Password not found in the wordlist.")
        elapsed_time = time.time() - start_time
        print(f"Total attempts: {attempts}")
        print(f"Total time: {elapsed_time:.2f} seconds")

    except Exception as e:
        print(f"[!] Unexpected error: {e}")

Explanation: The brute_force_cracker function iterates through WORDLIST, comparing each word to the target password. It tracks attempts, reports progress for large lists, stops if max_attempts is reached, and displays results.

Step 3: Parse Command-Line Arguments

Define a function to parse command-line arguments for the target password and optional settings.

def main():
    """
    Parse command-line arguments and run the brute-force cracker.
    """
    parser = argparse.ArgumentParser(description="Educational brute-force password cracker using an embedded wordlist.")
    parser.add_argument("-p", "--password", required=True, help="Target password to crack")
    parser.add_argument("-m", "--max-attempts", type=int, help="Maximum number of attempts (optional)")
    parser.add_argument("-i", "--progress-interval", type=int, default=10000, help="Show progress every N attempts (default: 10000)")
    
    args = parser.parse_args()
    
    brute_force_cracker(args.password, args.max_attempts, args.progress_interval)

Explanation: The main function uses argparse to parse the required -p (password) argument and optional -m (max attempts) and -i (progress interval) arguments, then calls brute_force_cracker.

Step 4: Run the Script

Start the script with error handling for user interrupts.

if __name__ == "__main__":
    try:
        main()
    except KeyboardInterrupt:
        print("\n[!] Stopped by user (Ctrl+C).")

Explanation: This block runs the main function and catches KeyboardInterrupt (Ctrl+C) to exit gracefully.
